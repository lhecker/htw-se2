### Requirements:
- [io.js](https://iojs.org/)

### Installation:
	npm install -g grunt-cli bower
	npm install
	bower install

### Usage:

###### Development
	grunt serve

###### Testing
	grunt test

###### Building a static standalone page
	grunt dist

###### Building standalone nw.js apps
	grunt build
